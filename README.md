# ☕ JAVA-OOPS

**Hands-on practice repo for learning Object-Oriented Programming concepts in Java.**

Small, focused `.java` files — each one built to understand one OOP idea at a time: classes, objects, and the different flavors of constructors.

---

## 📚 What's inside

| File | Concept covered |
|---|---|
| `basicoops.java` | Classes & objects basics — creating multiple objects, calling instance methods |
| `nperameterised.java` | Non-parameterized (default) constructor |
| `perameterised.java` | Parameterized constructor |
| `copy.java` | Copy constructor — creating a new object from an existing one |

---

## 🧠 Concepts practiced

- ✅ Classes & Objects
- ✅ Instance variables & methods
- ✅ The `this` keyword
- ✅ Constructors — default, parameterized, and copy
- ✅ Multiple objects of the same class behaving independently

---

## 🚀 Running the code

Each file is self-contained with its own `main` method. Compile and run any example directly:

```bash
javac basicoops.java
java basicoops
```

Swap `basicoops` for any other file name (`copy`, `perameterised`, `nperameterised`) to run that example.

> Requires JDK installed and available on your `PATH`.
